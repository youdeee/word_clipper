# Word Clipper
英単語をひたすらクリップするChrome Extension

## 使い方
1. Chromeでページ上のテキストをマウスで選択する
1. キーボードの a を押す
1. 選択領域が自動的に保存される
1. オプション画面で保存した単語リストを確認する
1. 単語に意味つけたり、いらない単語消したり、csv(?)形式で単語リストをダウンロードできる
